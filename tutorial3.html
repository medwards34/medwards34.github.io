Reference: Github user interface


The home page of Github offers a complete navigation through the website. Here is what each section lets you navigate to.

Profile area

Profile icon menu

  • Signed in as "username": Takes you to your profile page.

  • Set status: Lets other users see your current availability.

  • Your profile: Takes you to your profile page.

  • Your repositories: Takes you to your repositories page to view all your repositories.

  • Your codespaces: Takes you to cloud-based development environments page for organizations.

  • Your projects: Takes you to the projects portion of your profile page to create projects for your repositories.

  • Your stars: Allows you to create lists of repositories you follow.

  • Your gists: Takes you to the gist page where you can share snippets of your code to other users.

  • Upgrade: Takes you to upgrade tools or packages for your Github page.

  • Feature preview: Use this to see new features offered by Github and to enable or disable these features.

  • Help: Takes you to the docs page of Github where you can find help for the entire website.

  • Settings: Takes you to the settings page to change appearence, access, projects, and security for your account.

  • Sign out: Signs you out of your Github account.


Adding new projects

+ button menu

  • New repository: Click this to create a new repository.

  • Import repository: Click this to import old repositories onto your account.

  • New gist: Click this to create a new gist.

  • New organization: Takes you to the organization plan page to create a team or organization for your projects.

  • New project: Click this to create a new project.


Notifications

Bell Icon

  • Notifications: Hover the bell icon to see current notifications or click to get to the notifications page.

Navigation

Top menu bar

  • Github icon: Takes you to the homepage of your Github account.

  • Search bar: Type inside this bar to search for any repositories, projects, or users on Github.

  • Pull requests: Takes you to your pull requests to show others changes you made to your repository branches.

  • Issues: Takes you to the issues page where you can track ideas, bugs, and development of your repositories.

  • Marketplace: Takes you to the Github Marketplace for add-ons and application to extend your Github account.

  • Explore: Takes you to the explore page of Github to find trends, topics, and events happening through Github.


Activity

Side menu bar

  • Repositories: Displays your current repositories.
    • Click the "New" button above the search bar to create a new repository.
    • Type in the search bar to find any of your repositories.

  • Recent activity: Displays what recent activity you have been doing on your Github page.

Main page

Center area:

  • All activity: Displays all your activity on your Github main page including users and repositories you follow.

Website links

Bottom menu

  • Lightbulb icon: Provides tips for you about your main page.

  • RSS subscription: Provides a link to subscribe to the RSS feed.

  • Blog: Takes you to Github’s blog about updates with the company.

  • About: Takes you to Github’s about page to find out more about the company.

  • Shop: Takes you to Github’s store where you can purchase merchandise.

  • Contact Github: Takes you to Github’s contact page for help and resources with the website.

  • Pricing: Takes you to Github’s pricing page for purchasing packages to your Github account.

  • API: Takes you to Github’s docs page.

  • Training: Takes you to Github’s consultation page for help with projects and tailored training packages.

  • Status: Displays your current systems status on Github.

  • Security: Shares Github’s security measures and packages for project security.

  • Terms: Takes you to Github’s terms of service and directory on their docs page.

  • Privacy: Takes you to Github’s privacy statement on their docs page.

  • Docs: Takes you to Github’s docs page.

End of Page.
