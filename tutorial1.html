Creating a README.md on Github

Welcome to Github! Follow the steps below to create a Github account in order to create your first file: a README.md. While Github is not a social media website the README.md is one of the first and easiest ways to start using some of the many functions that Github has to offer their users. The README.md file will let other Github users know why you created a Github account and what kind of projects you will be working on. Think of it as an introduction file stored to your personal website.

Signing up

First sign up with Github.
Click the "Sign up" button on the upper right corner of your Github main page and follow the instructions.


You will need:

  • An email address.
  • A unique password for logging in.
  • A unique username that has not been used yet for logging in.

I. Access your README.md

Once you have signed up for your Github account you will be taken to the main page called the dashboard.
On the dashboard you will find a small box that displays a generic README.md for your profile.

  • Click the "Continue" button as seen in the picture below to edit your README.md.



II. Create the README.md

On the next page you will see your username prompted with the README.md file name as well as the section you can edit.

  • Scroll down to the editing portion of the README.md as seen in the picture below.
  • Edit your README.md to introduce yourself to other Github users and let them know why you have joined the website and what kind of projects you will be working on.

III. Committing your README.md

Once you have finished editing the README.md it is time to commit the file.

  • Scroll down the page to find the commit section of the README.md pictured below.
  • Add a description of your new file in the smaller text box. By not adding a description a default description will remain.
  • Add an extended description of your new file in the larger text box. By not adding a description it will remain blank.
  • Click the "Commit new file" button to commit the README.md to your profile.

IV. Accessing and editing your README.md

Congratulations! You have now created a README.md file and your first file on your Github account.

  • Access and edit your README.md at any time by going to your profile page seen below.
